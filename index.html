Stop The Rock

This is a 2D Arcade casual game with plenty levels, with simple intuitive controls, where the player must defend the castle and destroy the enemies...

  • Great attention is focused on optimization performance through optimizing scripts code, Object pooling, Scriptable object, UI canvas divided into static and dynamic.
  • Very simple and intuitive controls, with a clear user interface, even for the youngest ages of the players
  • Complex shop system with lots of items to buy and upgrades: Weapons, Armors, Castles, Protections, Treasures, in-game purchases
  • Enemies with different spawn position and firing patterns
  • Single player or two players.
  • Build for mobile Android and IOS
  • At the moment they are not implemented: some animations, FX & post-processing, particle and sound effects
  • To implement in the future: Boss-fight, Mini-game,

  • Made with Unity

Hiper Casual

Simple intuitive 2D game. Goal is to collect different shapes, and for certain combinations you get a bonus.

  • In the game we have: Entry screen, game play, Pause screen, Finish screen, Defeat screen
  • The entire user interface is responsive for all screen sizes mobile phones.
  • We have a random generation of different elements and random places from which they fall.
  • Falling speed of the elements increases dynamically depending on the time spent in the game.
  • The speed of generating elements increases dynamically also depending on the time spent in the game.
  • Animations, Particle Effect.
  • Saving System, high score.
  • Made for android phones.

  • Made with Unity

Cave Escape

This is a 2D Platformer / Dungeon crawler where the player must collect enough diamonds through the level to buy the Key of the Castle while avoiding enemies.

  • Player character animations, movement, looting, health, etc.

  • Three types of enemies, that inherit the same properties, but each has unique characteristics and behavior. The spider will shoot projectiles at the player's sight and skeleton will chasing a player in the field
  • Universal health system that works for both player and enemy
  • HUD Loot System & Shop UI (Player health, currency, mobile input, shop)
  • Crossplatform Input Windows and Android
  • Unity ads integration: players get extra coins to spend at the shop if they agree to watch the full-length ad
  • 2D Tilemap setup, parallax enviroment effect
  • To implement in the future: Boss-fight, puzzle elements...

  • Made with Unity
